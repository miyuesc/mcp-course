# 欢迎来到 🤗 模型上下文协议 (MCP) 课程

![MCP Course thumbnail](https://huggingface.co/datasets/mcp-course/images/resolve/main/unit0/1.png)

欢迎来到当今 AI 领域最激动人心的话题：**模型上下文协议 (MCP)**！

这门免费课程将带您踏上一段旅程，**从初学者到精通者**，理解、使用和构建 MCP 应用程序。

第一单元将帮助您入门：

* 了解**课程大纲**。
* **获取有关认证流程和时间安排的更多信息**。
* 认识课程背后的团队。
* 创建您的**账户**。
* **注册我们的 Discord 服务器**，与您的同学和我们见面。

让我们开始吧！

## 这门课程您能期待什么？

在这门课程中，您将：

* 📖 在**理论、设计和实践**方面学习模型上下文协议。
* 🧑‍💻 学习**使用已建立的 MCP SDK 和框架**。
* 💾 **分享您的项目**并探索社区创建的应用程序。
* 🏆 参与挑战，您将**评估您的 MCP 实现与其他学生的实现**。
* 🎓 通过完成作业**获得完成证书**。

还有更多！

在本课程结束时，您将了解**MCP 的工作原理以及如何构建利用外部数据和工具的 AI 应用程序，使用最新的 MCP 标准**。

不要忘记[**注册课程！**](https://huggingface.co/mcp-course)

## 课程是什么样的？

课程由以下部分组成：

* _基础单元_：您将在**理论上**学习 MCP **概念**。
* _实践操作_：您将学习**使用已建立的 MCP SDK** 来构建您的应用程序。这些实践部分将有预配置的环境。
* _用例作业_：您将应用所学概念来解决您选择的现实世界问题。
* _合作_：我们与 Hugging Face 的合作伙伴合作，为您提供最新的 MCP 实现和工具。
 
这**门课程是一个活跃的项目，随着您的反馈和贡献而发展！** 请随时在 GitHub 中提出问题和 PR，并在我们的 Discord 服务器中参与讨论。

## 先决条件是什么？

要能够跟上这门课程，您应该具备：

* 对 AI 和 LLM 概念的基本理解
* 熟悉软件开发原理和 API 概念
* 至少一种编程语言的经验（将展示 Python 或 TypeScript 示例）

如果您没有这些，不要担心！以下是一些可以帮助您的资源：

* [LLM 课程](https://huggingface.co/learn/llm-course/) 将指导您了解使用和构建 LLM 的基础知识。
* [代理课程](https://huggingface.co/learn/agents-course/) 将指导您使用 LLM 构建 AI 代理。

<Tip>

上述课程本身不是先决条件，所以如果您理解 LLM 和代理的概念，您现在就可以开始课程！

</Tip>

## 我需要什么工具？

您只需要 2 样东西：

* _一台计算机_ 和互联网连接。
* 一个_账户_：访问课程资源和创建项目。如果您还没有账户，可以在[这里](https://huggingface.co/join)创建一个（免费）。

## 认证流程

您可以选择以_旁听模式_跟随这门课程，或者做活动并_获得我们将颁发的两个证书之一_。如果您旁听课程，您可以参与所有挑战并做作业，**您不需要通知我们**。

认证流程是**完全免费的**：

* _获得基础知识认证_：您需要完成课程的第 1 单元。这适用于想要了解 MCP 最新趋势的学生，无需构建完整应用程序。
* _获得完成证书_：您需要完成用例单元（2 和 3）。这适用于想要构建完整应用程序并与社区分享的学生。
